Métodos de Arrays (1/4)

a) toString-> Unifica os registros do Array separando por virgula.
b) join -> Pega os valores do array e separa conforme voce passa no parametro.
c) indexOf -> Pega a posição do valor que esta no Array (Se o valor for igual a -1 resultado nao encontrado).
d) pop -> Remove o ultimo valor do array.
e) shift -> Remove o primeiro valor do array.
f) push -> Adiciona um valor no array ou altera os valor do array.

Métodos de Arrays (2/4)

g) splice -> Deleta o valor do array.
h) concat -> Junta (unifica lista + lista2) os valores dos arrays.
i) sort -> Ordena os valores em ordem alfabetica.
i) reverse -> Ordena os valores em ordem revertida.

Métodos de Arrays (3/4)

j) map -> permite percorrer um vetor e obter um novo array cujos itens são o resultado de uma função de callback que recebe como parâmetro cada item original.
k) filter -> O método filter() cria um novo array com todos os elementos que passaram no teste implementado pela função fornecida.
l) every -> O método every() testa se todos os elementos do array passam pelo teste implementado pela função fornecida.
m) some -> O método some() testa se ao menos um dos elementos no array passa no teste implementado pela função atribuída e retorna um valor true ou false.

Métodos de Arrays (4/4)

n) find -> O método find() retorna o valor do primeiro elemento do array que satisfizer a função de teste provida. Caso contrario, undefined é retornado.
